# Characters

Eight rigged characters from **Quaternius' "Ultimate Modular Men / Women"** packs.

- **Licence: CC0** (public domain). No attribution required, commercial use fine. <https://quaternius.com>
- Rigged, **no animation clips** — every pose (sitting, resting, clapping, hand-raising) is built by
  rotating bones in `../scene.js`.

To use your own instead: drop a rigged `.glb` in here and add its filename to `MODEL_FILES` in
`../scene.js`. Height, hip height and bone names are measured from the model at load, so it doesn't
have to match these — see "Using your own characters" in `../../README.md`.
